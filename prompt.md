# React E-Commerce Frontend

A responsive React E-Commerce application built using React, Vite, React Router DOM, and Context API.

## Features
- Product Listing
- Product Details
- Shopping Cart
- Cart Persistence (Local Storage)
- Guest Login
- Protected Checkout
- Responsive Design

## API
https://dummyjson.com/products

## Tech Stack
- React
- Vite
- React Router DOM
- Context API
- CSS
